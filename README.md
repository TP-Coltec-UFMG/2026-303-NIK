# Nik Is Kind
Um terrível vilão domina sua vila com sentimentos negativos e deixa todos os cidadãos vulneráveis!
Cabe a você salvar todos os membros da vila por meio de _minigames_ e se fortalecer para vencer
do terrível vilao!

## Características Gerais
Plataforma: PC

## Menu Inicial
O menu acessível consiste em:
- Jogar
- Opções
    - Narração
    - Filtros de daltonismo
- Sair
